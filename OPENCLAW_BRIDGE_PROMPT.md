# OpenClaw Bridge — Clean Build Prompt

## App Name
**OpenClaw Bridge**

## Visual Identity
- Use the provided OpenClaw mascot image as the primary App Store branding source.
- Style: minimalist, clean, flat accents.
- Palette: red, white, soft gray.
- Tone: friendly + professional.

## App Store Description
OpenClaw Bridge connects Telegram messages between iPhone and Apple Watch with real-time mirroring, smart filtering, and secure syncing.

## Feature Set
1. Real-time Telegram mirroring (private chats, groups, channels)
2. Apple Watch-first interactions (quick replies, haptics)
3. Secure relay layer with encrypted transport
4. Smart notification filters and priority tags
5. Multi-device sync (iPhone + Apple Watch, optional iPad/macOS)

## Technical Build Requirements
- Swift + SwiftUI for iOS app
- SwiftUI/WatchKit integration for watchOS app
- Telegram integration through Bot API or MTProto client
- Optional backend relay with Node.js or Python

## Deliverables
- iOS app shell with authentication, message list, filters, settings
- watchOS companion with glanceable messages + quick actions
- Relay layer for push/websocket sync
- Secure token handling and minimal retention
