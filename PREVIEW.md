# OpenClaw Bridge Preview

This repository now includes SwiftUI preview blocks so you can see screens instantly in Xcode canvas.

## iOS Preview
Open `ios/OpenClawBridge/MessageListView.swift` and run the **Inbox** preview.

What you should see:
- A red-accented "OpenClaw Bridge" navigation header
- A filter toggle (**Urgent only**)
- Message cards with priority pills (WORK / PERSONAL / URGENT)

## watchOS Preview
Open `watch/OpenClawBridgeWatch/WatchMessageListView.swift` and run the **Watch Inbox** preview.

What you should see:
- Compact chat list optimized for Apple Watch
- Red relay icon per row
- Secondary text for message body snippets
